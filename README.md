# reto3 Mörön
Repositorio para reto3 grupo Mörön

El repositorio está organizado en 2 carpetas, una para el modulo de lenguaje de marcas y el otro para programación.

La carpeta de programacion contiene:
Nuestro programa se llama Users.py en este caso y vamos a documentar y explicar nuestro código python. 

Este es nuestro programa principal y en él tenemos un menú con diferentes opciones en la que podemos añadir, eliminar y hacer inventario de las piezas como también consultar el inventario de las piezas, es decir, aquí podemos ver el inventario que van añadiendo al programa los usuarios tanto administradores como empleados. El programa está hecho con bucles en los que hemos definido en nuestro proyecto. Para crear el archivo csv por ejemplo, hemos inicializado la variable inicializar_csv y por si no existe nos crearia el archivo csv con su nombre, cantidad, piezas defectuosas y fecha de entrada

Por otra parte, tenemos otro programa que se llama tablas.py en el que básicamente hacemos el import de la librería sqlite3 para que nos cree la base de datos. Hemos usado la librería de pandas para compaginar sqlite y los archivos csv.
